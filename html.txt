Document Object Model (DOM): A tree of instructions that describes how the (html) elements are related to each other.

Common HTML Tags
<h1>A heading</h1>
<h2>Smaller heading</h2>
... <h6></h6>

Ordered List
<ol>
	<li>First item</li>
	           ...
	<li>N'th item</li>
</ol>

Unordered List
<ul>
	<li>First item</li>
	            ...
	<li>N'th item</li>
</ul>

Display an image
<img src="name.ext" alt="An example image" width="300">
Attributes:
-----> src: 'source'
-----> alt: 'alternative text'
-----> width: size (pixels)

Adding links
<a href="https://www.example.com">Click Here</a>
Attributes:
-------> href: Hipertext Reference

Adding tables
<table >
	<thead>
		<tr>
			<th>First heading</th>
			<th>Second heading</th>
			<th>Third heading</th>
		</tr>
	</thead>
	<tbody>
		<tr>
			<td>Data</td>
			<td>Data</td>
			<td>Data</td>
		</tr>
		<tr>
			<td>Data</td>
			<td>Data</td>
			<td>Data</td>
		</tr>
	</tbody>
</table>

Adding Forms
<form>
	<input type="text" placeholder="Example" name="example_name">
	<input type="submit">
</form>

Datalists
